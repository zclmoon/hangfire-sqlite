This is a simple template project that set up Hangfire with SQLite storage.

# Technique Stack & Components
1. ASP.NET Core 5.x
2. Hangfire
3. Hangfire.RecurringJobExtensions
4. Hangfire.Storage.SQLite
5. NLog.Web.AspNetCore
